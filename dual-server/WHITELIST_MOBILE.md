# Мобильные сети и Hysteria 2

В старой REALITY-схеме был отдельный whitelist-вход для мобильных операторов.
В Hysteria 2 версия проекта использует один UDP-вход с obfs salamander, поэтому
отдельный whitelist-конфиг больше не создаётся.

Проверьте:

1. В панели хостинга открыт UDP-порт Hysteria 2, обычно `443/udp`.
2. В UFW открыт тот же UDP-порт.
3. В клиент импортирована свежая `hysteria2://` ссылка.
4. Если оператор режет UDP 443, попробуйте `CLIENT_PORT=8443` или другой UDP-порт при установке.

Клиентские параметры:

```text
/etc/hysteria/hysteria-client-params.json
```
